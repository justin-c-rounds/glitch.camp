How do I use custom web fonts?

Custom web fonts allow us to have the browser download and use any fonts we want to use on our web pages, instead of having to rely exclusively on fonts installed on the user's device.

Using web fonts gives us the flexibility to fine-tune our typography, however it comes at a cost. Since a web font has to be downloaded to the user's device, adding a web font will increase the overall page weight and increase the number of requests the browser has to make to fully load the web page. This isn't much of a problem on devices with good network connections, but for devices on poor connections (like cell phone networks) this can make the web pages take too long to load. In addition, users who pay for data will be forced to use more bandwidth than necessary just to read the text on your page.

There are a few services which make it simple to use web fonts, and we'll explore each in detail.

Google Fonts

Google provides a web font service which makes it simple to use a custom font. Let's create a web page and apply a custom font to it using Google Fonts.

  1. We'll start with a basic web page:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Custom Web Fonts</title>
              </head>
              <body>
                <h1>Custom Web Fonts<h1>
              </body>
            </html>

  2. Let's add two p elements, one for the default font and one for the custom font:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Custom Web Fonts</title>
              </head>
              <body>
                <h1>Custom Web Fonts<h1>
                <p>The quick brown fox jumped over the lazy dog.</p>
                <p>The quick brown fox jumped over the lazy dog.</p>
              </body>
            </html>

  3. Go to Google Fonts, select a font, copy/paste the link tag provided into the head of your web page, and add the provided CSS to a custom class selector in a style element or external style sheet:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Custom Web Fonts</title>
                <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
                <style>
                  .custom {
                    font-family: 'Roboto', sans-serif;
                  }
                </style>
              </head>
              <body>
                <h1>Custom Web Fonts<h1>
                <p>The quick brown fox jumped over the lazy dog.</p>
                <p>The quick brown fox jumped over the lazy dog.</p>
              </body>
            </html>

    Now we add the custom class to the second p element:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Custom Web Fonts</title>
                <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
                <style>
                  .custom {
                    font-family: 'Roboto', sans-serif;
                  }
                </style>
              </head>
              <body>
                <h1>Custom Web Fonts<h1>
                <p>The quick brown fox jumped over the lazy dog.</p>
                <p class="custom">The quick brown fox jumped over the lazy dog.</p>
              </body>
            </html>
