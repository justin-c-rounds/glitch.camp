How do I create a fluid layout using absolute positioning?

We can use absolutely positioned elements to create a fluid layout which can size itself with respect to the browser window. This a fundamental principle of Responsive Design, and it will allow us to create a web page which functions predictably on mobile, tablet, and desktop browsers.

  1. We'll begin by creating a basic web page and adding some standard semantic elements to organize our layout:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
              </head>
              <body>
                <header>
                </header>
                <nav>
                </nav>
                <main>
                </main>
                <footer>
                </footer>
              </body>
            </html>

    The elements we are using are commonly found in web pages:

    • The header element represents a group of introductory or navigational aids. It may contain some heading elements but also other elements like a logo, a search form, and so on.
    • The nav element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
    • The main element represents the main content of the body of a document, portion of a document, or application. The main content area consists of content that is directly related to, or expands upon the central topic of, a document or the central functionality of an application.
    • The footer element represents a footer for its nearest sectioning content or sectioning root element. A footer typically contains information about the author of the section, copyright data or links to related documents.

  2. Right now these are empty, unstyled elements. So we'll add some placeholder text in each, and give them some background colors to make their boxes readily visible:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  header {
                    background: pink;
                  }
                  nav {
                    background: yellow;
                  }
                  main {
                    background: aqua;
                  }
                  footer {
                    background: lavender;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    Since we want our layout to work on mobile devices with small screens, we'll design it for a screen with a width of 320 pixels and a height of 480 pixels:

    Notice how the elements are positioned just inside the outer edge. This is because the body element has a default margin around it.

  3. The first thing we need to do is fix the body so it is the same size as the browser window (the outer black line in the example above):

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  body {
                    margin: 0;
                    position: absolute;
                    top: 0;
                    right: 0;
                    bottom: 0;
                    left: 0;
                  }
                  header {
                    background: pink;
                  }
                  nav {
                    background: yellow;
                  }
                  main {
                    background: aqua;
                  }
                  footer {
                    background: lavender;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    This removes the unwanted margin and sets each side of the body element to be flush against the edges of the browser window:

    With the body element positioned, we can now absolutely position all of layout elements within the browser window. For demonstration purposes, we'll work bottom-to-top, starting with the footer and finishing with the header.

  4. We'll position the footer at the bottom of the browser window:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  body {
                    margin: 0;
                    position: absolute;
                    top: 0;
                    right: 0;
                    bottom: 0;
                    left: 0;
                  }
                  header {
                    background: pink;
                  }
                  nav {
                    background: yellow;
                  }
                  main {
                    background: aqua;
                  }
                  footer {
                    background: lavender;
                    position: absolute;
                    right: 0;
                    bottom: 0;
                    left: 0;
                    height: 2rem;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    Notice we are not setting the top property. We are using the height property to determine the size of the footer. We want the footer to stretch horizontally, but not vertically as the browser window changes size. We are also using rem units for our height value so we can accommodate increases in font size.

  5. Now we'll position the main element just above the footer. It will take up most of the page, but we'll leave some space above it for the nav and header elements:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  body {
                    margin: 0;
                    position: absolute;
                    top: 0;
                    right: 0;
                    bottom: 0;
                    left: 0;
                  }
                  header {
                    background: pink;
                  }
                  nav {
                    background: yellow;
                  }
                  main {
                    background: aqua;
                    position: absolute;
                    top: 4rem;
                    right: 0;
                    bottom: 2rem;
                    left: 0;
                    overflow: auto;
                  }
                  footer {
                    background: lavender;
                    position: absolute;
                    right: 0;
                    bottom: 0;
                    left: 0;
                    height: 2rem;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    Notice we are setting the bottom property of the main element equal to the height of the footer element. Also we are not setting the height property because we want this element to stretch both horizontally and vertically.

    We also are adding overflow: auto since this is where all of the content is going to go, and we don't want the content overflowing the main element and messing up our layout. Setting the overflow to auto will tell the browser to automatically allow the content to scroll within the element if it exceeds the dimensions.

  6. Now we can position the nav above the main element, leaving room above it for the header:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  body {
                    margin: 0;
                    position: absolute;
                    top: 0;
                    right: 0;
                    bottom: 0;
                    left: 0;
                  }
                  header {
                    background: pink;
                  }
                  nav {
                    background: yellow;
                    position: absolute;
                    top: 2rem;
                    right: 0;
                    left: 0;
                    height: 2rem;
                  }
                  main {
                    background: aqua;
                    position: absolute;
                    top: 4rem;
                    right: 0;
                    bottom: 2rem;
                    left: 0;
                  }
                  footer {
                    background: lavender;
                    position: absolute;
                    right: 0;
                    bottom: 0;
                    left: 0;
                    height: 2rem;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    Just as we did with the footer, we set the height property on the nav, so it will only stretch horizontally:

  7. Finally we can position the header at the very top of the browser:

              
            <!DOCTYPE html>
            <html>
              <head>
                <title>Fluid Layout using Absolute Positioning</title>
                <style>
                  body {
                    margin: 0;
                    position: absolute;
                    top: 0;
                    right: 0;
                    bottom: 0;
                    left: 0;
                  }
                  header {
                    background: pink;
                    position: absolute;
                    top: 0;
                    right: 0;
                    left: 0;
                    height: 2rem;
                  }
                  nav {
                    background: yellow;
                    position: absolute;
                    top: 2rem;
                    right: 0;
                    left: 0;
                    height: 2rem;
                  }
                  main {
                    background: aqua;
                    position: absolute;
                    top: 4rem;
                    right: 0;
                    bottom: 2rem;
                    left: 0;
                  }
                  footer {
                    background: lavender;
                    position: absolute;
                    right: 0;
                    bottom: 0;
                    left: 0;
                    height: 2rem;
                  }
                </style>
              </head>
              <body>
                <header>HEADER</header>
                <nav>NAV</nav>
                <main>MAIN</main>
                <footer>FOOTER</footer>
              </body>
            </html>

    Now we have a fluid layout that will stretch to accommodate any screen width:
