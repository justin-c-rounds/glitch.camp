How do I make a web page?

  1. Create a file named index.html using your a text editor of your choice (e.g. Atom).
  2. Open the index.html in a browser of your choice (e.g. Firefox). Since the file is empty, the page will be completely blank.
  3. Back in the editor, start with a document type declaration: 
              
            <!DOCTYPE html>
  4. Add an html element (don't forget the closing tag!): 
              
            <!DOCTYPE html>
            <html>
            </html>
  5. Create a head element as a child of the html element: 
              
            <!DOCTYPE html>
            <html>
              <head>
              </head>
            </html>
  6. Create a body element as a sibling of the head element: 
              
            <!DOCTYPE html>
            <html>
              <head>
              </head>
              <body>
              </body>
            </html>
  7. Create a title element containing the title of your web page as a child of the head element: 
              
            <!DOCTYPE html>
            <html>
              <head>
                <title>My Web Page</title>
              </head>
              <body>
              </body>
            </html>
    This text will be displayed at the top of the browser tab/window and used in search results.
  8. Now start adding content to the body element. For example, we can add a 1st-order heading to our page to match the title text: 
              
            <!DOCTYPE html>
            <html>
              <head>
                <title>My Web Page</title>
              </head>
              <body>
                <h1>My Web Page<h1>
              </body>
            </html>
  9. Reload the index.html file in the browser. Continue adding more content in the editor and reloading your page in the browser to see the results.
  10. When you are satisfied with your web page, upload it to your host (e.g. Neocities).
